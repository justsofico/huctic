# Museum Exhibition Generator

Museum Exhibition Generator is a Python application that creates virtual museum exhibitions by combining artifacts from different historical periods and collections.

The project simulates the work of a museum curator by selecting exhibits, generating exhibition titles and producing a short catalog.

## Features

- Random exhibition generation
- Artifact catalog
- Theme selection
- Exhibition statistics
- Text catalog export

## Example

```
Exhibition

Machines That Changed History

Artifacts

• Antikythera Mechanism
• Roman Sundial
• Astrolabe
• Mechanical Clock
• Early Telegraph

Theme

Ancient Engineering

Artifacts

5

Historical Periods

4
```

Run

```bash
python main.py
```
